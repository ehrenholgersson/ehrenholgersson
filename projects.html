Some of My Projects

Shape Skirmish

Endless 2D platformer inspired by Geometry Dash / The Impossible Game. Name inspired by Geometry Wars because I always mix that up with Geometry Dash!

Originally started as something totally different at TAFE, but it got sidetracked and turned into its own thing. I plan to add some more polish, more terrain sections, sound, and a difficulty curve, then release on itch.io and google play to gain experience in the process of actually releasing something.

Picnic in the Park

Fairly simple Hangman game made for TAFE to demonstrate saving settings and game progress as well as use of canvas UI. 2D Game created with a 3D background made from Unity's terrain system (and a number of free assets from the asset store).

Untitled Space Game

2D game where you can build and fly custom space ships between different planets and (hopefully one day) solar systems.

This is a long term project that I have been working on for some time. I'm trying to focus my efforts on smaller things that can be built quickly to develop my portfolio right now but this game is my baby and I struggle to put it down for any length of time.

Currently you can build your own ships and fly between 3 planets, there are AI traders that fly between the planets and pirates that will attack them.

Multiplayer is partially implemented.

Smashformer

2D fighting game somewhat inspired by Smash Brothers, done as a group project in TAFE. All code by myself.

Solo Man's Quest
for the Volcano Idol

Another TAFE project. This was to demonstrate use of Unity NavMesh agents to navigate a maze. I went beyond the requirements of the brief to develop a full gameplay loop (although there is no player input as one of the nav agents assumes the protagonist/player role).

Our AI protagonist is tasked with navigating the level to reach his goal at the top of a volcano, avoiding a pursuing ogre and several patrolling spiders, deciding along the way if he is brave enough to risk grabing additional treasures in spite of the danger.

Assets from the unity store are used.
